About the Project

As part of high-tech studies in fullstack development course in technologies and web applications, we were required to develop an application of Criptonite coins and virtual currencies. this project makes information and reports from the world of virtual trade accessible. The application is SPA (Single Page App) that made up of diffrent package service functions. These functions wrap sub-functions for in order to perform various actions, to put the various functions in order to cause reading and synchronization between the different uses and efficiency. There is a use with config to make various keys in order to create a shortcuts to change in the future, according to personal need without looking for each function what it performs in example to change the coins cache live time . The application performs API readings in order to make information about virtual currency and its current price accessible according to ILS USD and EURO and the symbol of each coin. At the bottom of the application page I added a form to fill in the user's personal details and a switch button for choose a standard user or a premium user. If the user is not premium user, he can choose only 5 coins that he can follow. On the other hand the premium user can mark and track coins with no limit on the amount of coins he chooses. I decided to make a search line that by typing several letters, all the coins that contain the same typed letters in the coins name will appear automatically without the need for a search button. The data and history of the selected coins and user details saved in Local Storage area. I added as an attempt a button to change language. The application is adapted for all different resolutions for example mobile.
